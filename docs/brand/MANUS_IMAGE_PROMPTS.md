## SECTION 2 — AGENT PORTRAIT RE-GENERATION

The 65 portraits Manus sent have a **duplication problem** — same base face structure across multiple agents. Below are re-generation prompts that force clear visual differentiation. Use these for the agents most visible on the website.

### The Problem
Manus likely used a consistent base prompt with variations. This creates the "same person with different hair" problem. To fix it:
1. **Vary ethnicity aggressively** — specify nationality, not just generic "diverse"
2. **Vary age clearly** — young 20s vs experienced 40s vs seasoned 50s
3. **Vary background environment** — each agent should look like they're IN their domain
4. **Vary lighting temperature** — cold blue (tech) vs warm amber (creative) vs daylight (ops)
