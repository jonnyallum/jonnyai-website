# JonnyAI — The Digital Nexus

> **Main Website of the Antigravity Orchestra** | Powered by Jai.OS 4.0 | Next.js 15 | React 19 | Tailwind CSS v4

---

## 🏛️ Project Overview

This repository hosts the main digital storefront and nexus for **JonnyAI** and the **Antigravity Orchestra**. It showcases the 69 specialized agents, their live performance metrics, and the agency's "Collective Velocity" methodology.

## 🚀 Key Features

- **Live Agent Roster**: Dynamically synced from the Orchestra's local registry via `sync_website.py`.
- **God-Tier Aesthetics**: Built with Framer Motion and Tailwind CSS v4 for a premium, high-impact user experience.
- **Shared Brain Integration**: Direct feeds from Supabase for real-time wins, incidents, and performance data.
- **Glass Box Strategy**: Transparent project tracking for clients, demonstrating the inner workings of the Hive Mind.

## 🛠️ Technical Stack

- **Framework**: Next.js 15 (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS v4 (Industrial Theme)
- **Animations**: Framer Motion
- **Backend**: Supabase (PostgreSQL / Edge Functions)
- **Deploy**: Vercel
