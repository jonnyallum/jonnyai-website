# Forecast Scenario Modeling

## Description

Construction of multi-scenario forecasts (best, worst, likely) with documented assumptions, confidence intervals, and sensitivity analysis.

## Implementation Instructions

1. Define the core metrics and time horizon for the forecast.
2. Document all input assumptions (e.g., market growth, agent efficiency).
3. Build logic for best-case, worst-case, and base-case scenarios.
4. Assign confidence scores to each scenario based on data quality.
5. Perform sensitivity analysis to identify the most critical variables.

## Constraints

- **DO NOT** present a single-point forecast as certainty.
- **ALWAYS** include the "Assumptions Matrix" with every model.
- **DO NOT** project linearly without accounting for market volatility.
