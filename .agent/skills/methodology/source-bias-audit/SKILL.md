# Source Bias Audit

## Description

Systematic analysis of source sets for bias patterns (economic, geographic, ideological) to ensure objective intelligence.

## Implementation Instructions

1. Profile current source sets for regional or platform-specific bias.
2. Audit extracted claims for recurring narrative patterns.
3. Diversify source sets to include opposing views or independent data.
4. Document bias risks in research reports to inform stakeholders.
5. Continuously re-evaluate source trust based on bias outcomes.

## Constraints

- **DO NOT** treat high-authority sources as bias-free by default.
- **ALWAYS** disclose known bias risks in research outcomes.
- **DO NOT** use source sets with a single point of ideological failure.
