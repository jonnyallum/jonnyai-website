# Platform Selection Matrix

## Description

Course platform evaluation — systematic comparison of Teachable, Kajabi, Circle, Thinkific, and custom builds against course requirements.

## Implementation Instructions

1. Define platform requirements: student volume, community features, payment, drip content, branding.
2. Score each platform against requirements using a weighted matrix.
3. Assess total cost: platform fee plus transaction fee plus migration cost.
4. Evaluate migration risk: how hard is it to move students and content if the platform changes.
5. Confirm technical requirements: compliance, certificates, API access, webhooks.
6. Produce a recommendation with rationale and a migration risk caveat.

## Constraints

- DO NOT recommend a platform without a cost projection for 12 months.
- ALWAYS include a custom build option with realistic development cost estimate.
- DO NOT select based on brand familiarity alone — evaluate against defined requirements.
