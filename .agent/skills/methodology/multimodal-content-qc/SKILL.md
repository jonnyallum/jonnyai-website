# Multimodal Content QC

## Description

Quality-control process for text, image, and video assets to ensure factual coherence, brand consistency, and compliance.

## Implementation Instructions

1. Validate cross-modal message alignment (script, visuals, CTA).
2. Check factual claims and on-screen text accuracy.
3. Audit brand tone, design tokens, and accessibility cues.
4. Flag mismatches and generate precise correction notes.
5. Approve only after all critical defects are resolved.

## Constraints

- **DO NOT** pass assets with contradictory text/visual claims.
- **ALWAYS** run accessibility checks on published assets.
- **DO NOT** approve unresolved critical compliance issues.
