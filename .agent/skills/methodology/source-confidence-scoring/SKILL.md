# Source Confidence Scoring

## Description

Standardizes source trust evaluation using provenance, recency, authority, and corroboration.

## Implementation Instructions

1. Define confidence rubric (Primary, Secondary, Tertiary, Unknown).
2. Score each source by authority, date, bias risk, and traceability.
3. Require corroboration for medium/low-trust sources.
4. Attach confidence tags to all research outputs.
5. Continuously update trusted-source registry from outcomes.

## Constraints

- **DO NOT** treat all sources as equal quality.
- **ALWAYS** disclose confidence level in final deliverables.
- **DO NOT** remove provenance links during summarization.
