# Audience Segmentation Strategy

## Description

Method for creating and refining audience segments for creative, channel, and offer tailoring to improve marketing efficiency.

## Implementation Instructions

1. Extract audience data (Demographic, Behavioral, Psychological).
2. Group into distinct "High-Value" and "Efficiency" segments.
3. Map each segment to a specific offer-hook and creative style.
4. Track performance (CPA, ROAS) at the segment level.
5. Continuously update segment definitions based on outcome data.

## Constraints

- **DO NOT** create segments so small they lack statistical scale.
- **ALWAYS** define segments based on actionability (how can we target them?).
- **DO NOT** use static segments; automate the refresh based on behavior.
