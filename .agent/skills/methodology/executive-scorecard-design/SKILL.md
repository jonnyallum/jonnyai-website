# Executive Scorecard Design

## Description

Executive-level scorecard design — 3 KPI maximum, strategic focus, mobile-first layout for C-suite decision makers.

## Implementation Instructions

1. Confirm dashboard type is Strategic: exec audience, 3 KPIs max, no drilldowns.
2. Select the 3 highest-decision-impact KPIs from the DASHBOARD_SPEC.md.
3. Design mobile-first: vertical KPI stack, minimum 48px touch targets, no horizontal scrolling.
4. Apply RAG (Red/Amber/Green) status to each KPI against defined thresholds.
5. Embed a 4-week trend sparkline under each primary KPI number.
6. Brief @priya on branded color application and typography scale.

## Constraints

- DO NOT exceed 3 KPIs on an executive scorecard.
- ALWAYS design mobile-first — the primary layout is phone-sized.
- DO NOT add filters, drilldowns, or secondary metrics to a Strategic dashboard.
