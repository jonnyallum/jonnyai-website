# Curriculum Backwards Design

## Description

Backwards design methodology — building curriculum from learning outcomes backwards to module structure, content, and assessments.

## Implementation Instructions

1. Define the terminal learning outcomes: what can students DO after completing the course.
2. Map assessments: for each outcome, define how mastery will be demonstrated.
3. Sequence modules in reverse: what must be learned before the final outcome is achievable.
4. Draft module objectives: each module has one primary, testable learning outcome.
5. Assign content format to each module (video, text, interactive, live).
6. Review the sequence for logical flow and prerequisite dependency.

## Constraints

- DO NOT begin content creation before outcomes and assessments are defined.
- ALWAYS ensure each module objective is directly traceable to a terminal outcome.
- DO NOT add modules without a justified learning outcome — no padding.
