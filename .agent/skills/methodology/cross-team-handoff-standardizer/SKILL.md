# Cross-Team Handoff Standardizer

## Description

Standardized framework for handoff artifacts, ownership transfer, and accountability between agents and teams.

## Implementation Instructions

1. Define the mandatory "Handoff Packet": Goal, Progress, Blocker, Artifacts, NEXT_STEP.
2. Implement handoff-validation (Receiver confirms they have everything needed).
3. Standardize handoff events in the Agency Chatroom.
4. Audit for "dropped balls" (stale tasks with no clear handoff).
5. Refine the standardizer based on team-friction reports.

## Constraints

- **DO NOT** hand off tasks without a verified "Next Step."
- **ALWAYS** ensure the receiver is mission-ready (has skills/tools).
- **DO NOT** use ambiguous language in handoff summaries.
