# Social Proof Experimentation

## Description

Technique for placing and testing different social proof elements (reviews, count, certifications) to optimize conversion confidence.

## Implementation Instructions

1. Collect diverse social proof assets (Case studies, Logos, Counter).
2. Select high-friction pages/sections for placement.
3. A/B Test variants: "The Expert" (Certifications) vs "The Crowd" (Counter) vs "The Human" (Quotes).
4. Measure uplift in "Confidence Actions" (e.g., Click to See Pricing, Buy Now).
5. Automate the display of highest-performing proof for each audience segment.

## Constraints

- **DO NOT** use fabricated or misleading social proof.
- **ALWAYS** link social proof to verified sources where possible.
- **DO NOT** allow social proof to clutter the UI or hide CTAs.
