# Course Sales Funnel Design

## Description

Funnel architecture for course landing pages, email sequences, waitlists, and conversion optimisation.

## Implementation Instructions

1. Define the funnel entry point: cold traffic, warm audience, or email list.
2. Design the landing page structure: transformation promise, proof, curriculum, offer, CTA.
3. Build the pre-launch waitlist sequence: 5-7 email warm-up with value content.
4. Create the launch sequence: 5-day open cart email campaign with urgency mechanics.
5. Design the post-purchase onboarding: confirmation, access, and quick-win first lesson.
6. Set up conversion tracking: landing page to opt-in to purchase at minimum.

## Constraints

- DO NOT use false scarcity or countdown timers that reset — one genuine urgency mechanism only.
- ALWAYS include a money-back guarantee statement to reduce purchase risk.
- DO NOT skip post-purchase onboarding — the first 48 hours determine long-term retention.
