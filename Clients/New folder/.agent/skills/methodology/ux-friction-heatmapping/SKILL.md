# UX Friction Heatmapping

## Description

Mapping of user journey friction points through session analysis, click-tracking, and frustration-signal detection.

## Implementation Instructions

1. Instrument key journeys with click-tracking and session recording.
2. Identify "Rage Clicks," "Dead Clicks," and "Excessive Scrolling."
3. Build a friction heatmap for the target flow.
4. Prioritize fixes based on friction severity and business impact.
5. Verify friction reduction with post-fix behavioral analysis.

## Constraints

- **DO NOT** ignore friction signals from mobile users.
- **ALWAYS** protect user privacy in session recording data.
- **DO NOT** mistake high engagement for low friction without conversion data.
