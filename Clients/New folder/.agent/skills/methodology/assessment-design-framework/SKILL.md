# Assessment Design Framework

## Description

Quiz, project, and assessment design aligned to learning outcomes — Bloom's taxonomy application, rubric creation, and automated grading design.

## Implementation Instructions

1. Map each assessment to a specific module learning outcome — no orphan assessments.
2. Classify cognitive level using Bloom's taxonomy: remember, understand, apply, analyse, evaluate, create.
3. Design quiz questions at the appropriate cognitive level — avoid trivial recall for advanced outcomes.
4. Define a project brief with a rubric: clear criteria, scoring bands, and exemplar outputs.
5. Automate grading where possible: multiple choice auto-scores, project rubrics guide manual review.
6. Pilot assessments with 3-5 students before deployment to check for ambiguity.

## Constraints

- DO NOT create assessments without a direct link to a learning outcome.
- ALWAYS provide a rubric with at least 3 scoring levels for any subjective assessment.
- DO NOT use trick questions — assessments measure mastery, not gotcha moments.
