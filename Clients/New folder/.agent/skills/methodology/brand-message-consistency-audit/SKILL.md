# Brand Message Consistency Audit

## Description

Cross-touchpoint audit of brand voice, messaging hierarchy, and copy consistency to ensure unified positioning.

## Implementation Instructions

1. Inventory all active customer touchpoints (Site, Ad, Social, Email).
2. Extract core messages and voice samples from each.
3. Audit against the "Global Brand Voice" and "Positioning Statement."
4. Flag "Message Dilution" or voice-shifts in fringe channels.
5. Generate a "Consistency Report" and update outdated copy.

## Constraints

- **DO NOT** allow conflicting "tone of voice" across different platforms.
- **ALWAYS** check for consistency in terminology and naming conventions.
- **DO NOT** ignore microcopy (buttons, error states) in the audit.
