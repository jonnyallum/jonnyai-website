# Benchmark Intelligence Synthesis

## Description

Method for synthesizing complex competitor benchmark data into high-level, actionable insights and strategic recommendations.

## Implementation Instructions

1. Standardize benchmarking metrics across competitors (e.g., pricing, features, speed).
2. Rank competitors into tiers based on performance segments.
3. Identify specific "winning" playbooks and "losing" patterns.
4. Synthesize into a "State of the Market" report for leadership.
5. Link benchmarks to specific growth or product opportunities.

## Constraints

- **DO NOT** provide raw data tables without synthesized conclusions.
- **ALWAYS** include "Next Steps" with every intelligence report.
- **DO NOT** reuse outdated benchmarks for active decision cycles.
