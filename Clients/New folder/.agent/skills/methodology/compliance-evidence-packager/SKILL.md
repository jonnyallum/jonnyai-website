# Compliance Evidence Packager

## Description

Builds audit-ready evidence packs for legal, security, and regulatory reviews across systems and workflows.

## Implementation Instructions

1. Define evidence scope by framework or audit request.
2. Collect artifacts: policies, logs, approvals, and controls evidence.
3. Validate completeness, timestamps, and traceability links.
4. Package evidence in standardized index and folder structure.
5. Deliver signed checklist and gap report to reviewers.

## Constraints

- **DO NOT** submit evidence without provenance metadata.
- **ALWAYS** include control-to-evidence mapping.
- **DO NOT** alter raw logs during packaging.
