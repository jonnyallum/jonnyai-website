# Module Completion Optimization

## Description

Engagement optimisation for course modules — improving retention, reducing drop-off, and increasing checkpoint completion rates.

## Implementation Instructions

1. Pull completion and drop-off data per module from the course platform.
2. Identify drop-off hotspots: any module with over 20% drop-off is a candidate for redesign.
3. Classify drop-off cause: content length, difficulty spike, format fatigue, or technical issue.
4. Apply the appropriate fix: shorten videos, add checkpoints, change format, or reduce friction.
5. Add retention hooks: end each module with a teaser of the next module value.
6. Re-measure after 2 cohorts and compare completion delta to baseline.

## Constraints

- DO NOT redesign based on fewer than 20 student completions — insufficient sample.
- ALWAYS test one fix at a time to isolate the improvement cause.
- DO NOT add gamification elements without confirming they align with the course audience.
