# Market Sentiment Fusion

## Description

Technique for merging news, social media, and industry reports into unified strategic sentiment analysis for accurate decision-making.

## Implementation Instructions

1. Standardize sentiment scoring across different source types.
2. Weigh sentiment by source authority and recency.
3. Correlate sentiment shifts with external events/market data.
4. Aggregate into a "Market Pulse" dashboard with trend indicators.
5. Feed sentiment data into growth and branding workflows.

## Constraints

- **DO NOT** use raw count data as sentiment without weight.
- **ALWAYS** disclose the source mix for fused sentiment reports.
- **DO NOT** ignore localized sentiment shifts for global brands.
