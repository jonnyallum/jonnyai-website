# Perplexity Output — Research Division

This folder is Perplexity's dedicated workspace in the Antigravity Orchestra repo.

All Perplexity research, agent drafts, intel, and ideas land here — never copy-pasted, always committed.

## Structure

```
perplexity/
├── research/   ← market intel, trend analysis, web research
├── agents/     ← new agent SKILL.md drafts
├── intel/      ← competitor profiles, pricing, OSINT
├── ideas/      ← venture concepts, monetisation blueprints
└── briefs/     ← client strategy docs
```
